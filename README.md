# BlogNettySocket
基于netty-socketio实现

## 核心依赖
``` xml
		<dependency>
			<groupId>com.corundumstudio.socketio</groupId>
			<artifactId>netty-socketio</artifactId>
			<version>1.7.7</version>
		</dependency>
```

## 基础配置

``` yml
# socket服务端配置
club.dlblog.socketio:
        host: localhost #IP地址
        port: 9000     #端口号
```
